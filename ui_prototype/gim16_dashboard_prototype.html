1. Pick the number of human tables and assign each one to a country.
2. Enter a short natural-language command per table, for example: “Increase AI spending moderately but avoid trade escalation.”
3. Launch the hybrid round and wait for the artifacts.

5. Let each table review the new world state and type the next command in the same input field for the next round.
